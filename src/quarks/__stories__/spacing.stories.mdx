# Spacing

## TODO

Cursor in thc have four defined quarks:

-   `--thc-spacing--XS`: 5px
-   `--thc-spacing--M`: 10px
-   `--thc-spacing--XL`: 20px
-   `--thc-spacing--XXL`: 30px
-   `--thc-spacing--4XL`: 50px
