# Spacing

## TODO

Cursor in thc have four defined quarks:

-   `--thc-spacing--XS`: 5px
-   `--thc-spacing--S`: 10px
-   `--thc-spacing--M`: 20px
-   `--thc-spacing--L`: 30px
-   `--thc-spacing--XL`: 50px
