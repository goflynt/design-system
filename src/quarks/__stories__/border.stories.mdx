# Borders

## Use border utilities to quickly style the border and border-radius of an element. Great for images, buttons, or any other element.

### Borders

Border in thc have three defined quarks:

-   `--thc-borderSolid--S`
-   `--thc-borderSolid--L`
-   `--thc-borderDashed--S`

They can be used with radius and color quarks.

By default the border will use the `--thc-textColor--secondary` for its color.

### Radius

Radius in thc have four defined quarks:

-   `--thc-radius--XS`: 3px
-   `--thc-radius--S`: 5px
-   `--thc-radius--M`: 10px
-   `--thc-radius--L`: 100px
