# Actionable

## TO RESUME

Actionable is an atom composed only with CSS, it exposes the following behaviors:

-   hover
-   active
-   disabled

The CSS object leverages `:hover`, `:active` and `[disabled]` pseudo selectors.

It will handle a different strategy to the lighter/darker effect.

Apply the following CSS selector to your component to add actionable behaviors to it:

-   `thc-o-actionable`
